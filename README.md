# TEAM NO.145-AuthShield: A Full Stack Authentication System using Spring Boot, React, Email OTP, and JWT for Secure Web Application Access
## About

AuthShield is a full stack authentication and authorization system designed to provide secure, scalable, and reliable user access control for modern web applications.
The system uses Spring Boot for backend services, React for frontend user interaction, Email-based OTP verification for identity validation, and JWT (JSON Web Tokens) for stateless authentication.
AuthShield ensures that only verified users can access protected resources while maintaining high security standards and a smooth user experience.

## Methodology
### 1. User Registration & OTP Verification

i) User registers using email and password.

ii) Password is encrypted using BCrypt.

iii) An OTP is generated and sent to the userâ€™s email.

iv) Account is activated only after successful OTP verification.

### 2. Authentication & JWT Generation

i) User logs in with valid credentials.

ii) Spring Security authenticates the user.

iii) A JWT token is generated upon successful login.

iv) JWT is sent to the client and stored securely.

### 3. Authorization & Secure API Access

i) Each request includes the JWT token in the header.

ii) Backend validates the token.

iii) Access is granted based on user role and permissions.

iv) Unauthorized access is blocked automatically.
